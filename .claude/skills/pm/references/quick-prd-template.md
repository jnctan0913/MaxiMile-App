# PM Reference: Quick PRD Template

Use this template for **QUICK tier** or when the user says "quick" or "skip discovery".

Quick mode skips:
- Stakeholder mapping
- Competitive analysis
- Customer journey
- Full discovery

Focus on: problem, features, metrics only.

---

## Quick PRD Template

```markdown
# Quick PRD: [Project Name]

## Problem
[One paragraph describing the core problem, the user affected, and the friction they experience]

## Solution
[Core features only -- what we will build to address the friction]

## Success Metric
[One metric that defines whether this solution is working]

## P0 Features (RICE)
| Feature | Score | Acceptance Criteria |
|---------|-------|---------------------|
| | | |

## Constraints
[Key constraints -- technical, business, or resource limitations]

## Risks
[Top 2-3 risks that could prevent success]
```
